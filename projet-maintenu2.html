Bot discord multi-fontions

Le projet est un bot discord qui vise à devenir le seul bot dont vous aurez besoin sur votre serveur. Encore en phase de développement, le bot a pour objectif d'implémenter toutes sortes de fonctions, en allant de la musique, en passant par les memes et le ban d'utilisateurs.

Plus sur le projet

Le répertoire du projet existe depuis Mars 2021 et son propriétaire est Jasper Mayone ( profil Github), et le bot est opérationnel depuis. Au fur et à mesure du temps, des fonctionnalités se sont rajoutés, comme par exemple les niveaux de permissions dans le serveur ou encore les paramègtres privés du serveur. Un site web est aussi disponible, et montre ce que le bot pourra faire dans le futur. Lien ici.


La maintenance réalisée sur ce projet est un système de publication automatique des messages dans les salons d'annonces sur un serveur. Au lieu de devoir appuyer sur le bouton "publier" sur discord, l'utilisateur peut saisir une commande publish <message ID/link> qui va publier le message au nom du bot sur le serveur.

Recherche liées au projet

Heptagram

Heptagram est le bot sur lequel j'ai contribué à résoudre l'issue #384. J'ai trouvé ce bot en cherchant des projets de bots opensources sur internet, et celui là m'a paru intéressant.

Publisher

Publisher est un bot discord qui fait de l'auto-publication de messages. C'est la principale source d'inspiration du code que j'ai fournis. Le bot fonctionne avec une base de données pour gérer plusieurs salon à traquer pour la publication de messages, point qui ne figure pas dans mon code car je n'avais aucune expérience dans le développement de bot discord.

Discord.js

Étant novice dans le développement en discord.js, je me suis beaucoup aidé de la documentation pour savoir quelles propriétés utiliser dans mon code (notament pour récupérer les informations sur un message, comme son auteur, son salon ou encore les permissions de son auteur.

Enmap

Enmap est un gestionnaire de données qui simplifie l'utilisation de la gestion de données pour les nouveaux utilisateurs de Javascript, et qui ne sont peut-être pas familiés avec les requêtes SQL. Heptagram implémente Enmap comme gestionnaire de données pour les settings d'un serveur notamment.
