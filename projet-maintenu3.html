Projet tutoré : PacMan

Le projet est le projet tutoré du deuxièeme semestre de DUT informatique, dans lequel nous devions reproduire un pacman en Java, à l'aide de JavaFx.

Plus sur le projet

Le projet a été réalisé en équipe de 5, et le but était de recréer un pacman et de lui rajouter ou non des fonctionnalités. Avec mon groupe, nous avions refait le pacman le plus précisément possible. Nous avons aussi rajouté un créateur de niveau, pour que le joueur puisse se mettre au défi avec d'autres maps.


La maintenance réalisée sur ce projet est le transfert de l'enregistrement des scores d'un fichier texte à une base de donné en local. Pour lier la base avec le jeu j'ai utilisé JDBC (Java DataBase Connection) et une base de donnée PostgreSQL.

Documentation

Lien github

Le répertoire github du projet. Il est issu d'un fork avec le projet principal et la branche dans laquelle j'ai effectué les modifications est la branche "maintenance". J'ai cré un fichier "ConnectionBDD.java" dans lequel j'ai regroupé toutes les classes dont j'avais besoin pour faire du DAO sur la base de données. J'ai ensuite modifié les fichiers dans lesquels les scores étaient mis à jour ou utilisés. Pour lier la base de données, il faut modifier les valeurs dans la méthode connect() de ConnectionBDD.
